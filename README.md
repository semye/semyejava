# java基础

## IO

### 字符

- ASCII码 一个英文字符占用一个字节
- Unicode编 一个英文字符或一个汉字字符都占用两个字节的空间
- UTF-8编码 一个英文字符占用一个字节，一个汉字字符占用三个字节的空间

### 字符操作类
